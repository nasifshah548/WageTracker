# WageTracker 🧾💼

**WageTracker** is a modern, responsive, and beginner-friendly React app that helps users easily track their daily working 
hours and earnings. Whether you're a freelancer, contractor, hourly worker, or just like to keep a record of your income, 
this app simplifies the entire wage tracking process.

---

## ✨ Features

- ✅ **Daily Input Tracking**: Log daily working hours and hourly rate.
- 📅 **Weekly Summary**: View total hours worked and earnings for the current week.
- 🗓️ **Monthly Summary**: Get a full overview of the hours and wages per month.
- 📆 **Yearly Summary**: Track your annual work performance and earnings.
- 🔐 **Local Storage Powered**: All entries are saved securely in your browser — no sign-up or login required!
- 📱 **Responsive UI**: Clean and elegant interface built with Tailwind CSS. Looks great on all devices.
- 🌍 **International Use**: Can be used by users around the world thanks to the date-fns library.
- 🕒 **Real-Time Updates**: All data updates in real-time as you log your hours.

## 🚀 Getting Started

Follow these steps to clone and run the app locally:

### 1. Clone the repository

```bash
git clone https://github.com/your-username/wage-tracker.git
cd wage-tracker


2. Install dependencies

Make sure you have Node.js and npm installed.

3. Run the app

npm start

The app will start at http://localhost:3000.

🛠️ Technologies Used

- React JS – for building UI

- Tailwind CSS – for styling

- date-fns – for date manipulation

- LocalStorage API – for saving user data locally
